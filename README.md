# Pokesearch

A Pokédex web app built with vanilla JavaScript, HTML and CSS.  
Fetches live data from [PokéAPI](https://pokeapi.co).

## Features

- Browse and search Pokémon by name
- Filter by type
- Click any card for full stats, sprites and details
- Save favorites (persisted in localStorage)
- Parallax hero section

## Tech

- Vanilla JS / HTML / CSS — no frameworks
- [PokéAPI](https://pokeapi.co) for Pokémon data and artwork

## Run locally

Just open `index.html` in a browser — no build step required.
